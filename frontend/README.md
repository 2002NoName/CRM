# CRM Frontend

Prosty frontend z dashboardem oraz listą klientów, sprzedaży oraz pracowników.

## Technologie

- React.js
- React Router DOM
- Tailwind CSS
- Recharts

## Jak zacząć

1. Przejdź do katalogu `frontend`
2. Zainstaluj zależności: `npm install`
3. Uruchom aplikację: `npm start`
